# JavaScript

## JavaScript Array Methods 

| **Method**    | **What It Does**                                                      |
| ------------- | --------------------------------------------------------------------- |
| **`push()` **     | Adds an element to the end of the array                               |
| `pop()`       | Removes the last element from the array                               |
| `shift()`     | Removes the first element of the array                                |
| `unshift()`   | Adds an element to the beginning of the array                         |
| `map()`       | Creates a new array by applying a function to each element            |
| `filter()`    | Returns a new array with elements that pass a condition               |
| `reduce()`    | Reduces the array to a single value (sum, product, etc.)              |
| `forEach()`   | Executes a function for each array element (no return)                |
| `find()`      | Returns the first element that matches a condition                    |
| `findIndex()` | Returns the index of the first matching element                       |
| `includes()`  | Checks if the array contains a value                                  |
| `indexOf()`   | Returns the index of a value (or `-1` if not found)                   |
| `slice()`     | Returns a portion of the array without modifying it                   |
| `splice()`    | Adds/removes elements and modifies the original array                 |
| `sort()`      | Sorts the array (alphabetically or numerically with compare function) |
| `reverse()`   | Reverses the array order                                              |
| `concat()`    | Combines two or more arrays                                           |
| `join()`      | Converts array elements into a string with a separator                |
| `flat()`      | Flattens nested arrays to a specified depth                           |
| `flatMap()`   | Maps and flattens the array in one step                               |
